# Lava Lublin

Restaurant website for Lava Lublin, a local restaurant in Lublin, Poland. Designed with modern animations and a responsive layout.

Live at [lava-lublin-nextjs.vercel.app](https://lava-lublin-nextjs.vercel.app)

## What it does

- Full restaurant website with menu, location, and contact information
- Integrated live Google Reviews pulled directly from Google Maps API
- Instagram feed integration showing latest posts
- Embedded Google Maps for location and directions
- Smooth scroll animations and modern UI transitions
- Fully responsive across all screen sizes

## Tech stack

Next.js 15, React, TypeScript, Tailwind CSS, Google Maps API, Vercel
